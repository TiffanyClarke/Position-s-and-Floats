Activity 2: Floating

The float CSS property specifies that an element should be placed along the left or right side of its container, allowing text and inline elements to wrap around it. The element is removed from the normal flow of the web page, though still remaining a part of the flow (in contrast to absolute positioning).

In this activity, you are to achieve the following with as little extraneous code as possible:

  • Float the first paragraph to either the left or right so that it is visibily sitting beside the second paragraph (as if you were wrapping text around an image)
  • After completing the first part of this activity, fix the third paragraph so that it is no longer wrapping and it sites entirely below the first two paragraphs.
