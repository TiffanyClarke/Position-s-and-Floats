Activity 3: Columns

I've already added a parent div to each column group with a class of "clearfix" (check out the style block on this page). It acts like a regular CSS clear but it affects parent element containing the floats, causing the affected element to act is if they are cleared.

In this activity, you are to achieve the following with as little extraneous code as possible:

    ;
  1. Float two elements of equal width side-by-side using relative units. Add a margin to create a gutter (add space) between columns, again using relative units. (This will require some math and you will have to remember the box model!)
  2. Float three elements of equal width and add gutters, as above.
  3. Float four elements of equal width and add gutters, as above.
  4. Just so you can see what it does, try removing the .clearfix class from column group 1 and watch what happens! Once you see the effect, replace the .clearfix class so everything is working again!
