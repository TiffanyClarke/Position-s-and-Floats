Activity 1: Positioning

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).

In this activity, you are to achieve the following on each of the numbered elements with as little extraneous code as possible:

  1. Relative Positioning (.child-top-left):
    Position the element in the top left corner of the parent element.
  2. Absolute Positioning (.child-bottom-right):
    Position the element in the bottom right corner of the parent element.
  3. Fixed Positioning (.child-fixed-banner):
    Position the element so that it is fixed to the top right of the body element.
